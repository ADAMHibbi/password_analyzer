🔐 Advanced Password Pattern Generator & Similarity Analyzer

Description:

A Python-based cybersecurity tool designed to simulate real-world password weaknesses by generating and analyzing common password patterns. The tool intelligently predicts up to 1000 potential password combinations based on user-related inputs such as names, dates, and keywords.

It enforces realistic constraints by limiting generated passwords to 8–9 characters, reflecting common password policies used in modern systems.

The system also includes a similarity analysis engine, which evaluates how closely a user-defined password matches predictable patterns. Using pattern comparison techniques (such as positional matching or distance-based scoring), it identifies weak structures and returns the Top 10 closest matches with similarity percentages, providing clear insight into password strength.

Key Features:

- Generates up to 1000 realistic password patterns
- Enforces 8–9 character length policy
- Accepts structured and optional inputs (name, birth date, keywords)
- Detects if a password is predictable or common
- Computes similarity scores (%) against generated patterns
- Returns Top 10 closest password matches
- CLI-based interface with cybersecurity-style design
- Option to export results for further analysis
